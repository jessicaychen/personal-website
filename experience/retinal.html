    Bottom Layer Retinal Segmentation

    Advanced Robotic Eye Surgery Lab @ UCLA

    August 2019 - March 2020

    Background

    For precise semi-automated retinal injection in surgery, it is necessary for surgeons to know the depth of the bottom layer of the retina at any point they select in their field of view. The purpose behind the project was: Given a user-selected input of (x,y) coordinates from a top-down view of the retina from an scan, determine the approximate z-coordinate of the bottom layer of the retina through OCT volume scan image segmentation.

    Project

    I developed a functional algorithm using MATLAB to detect the bottom edge of the "retina" in a pig-eye model in OCT volume scans. The OCT scans are imported as intensity matrices, and I pre-processed the scans using filters to denoise the scans. A difficulty with this project is the fact that the ROI is the bottom layer, not the top layer; I got around this by first detecting the top layer and blacking it out. The following techniques were used:

      Pre-processing
      • Truncation - smaller ROI for efficiency
      • Averaging consecutive scans - smoothes an image and denoises
      • Sobel filter - emphasizes edges using two 3x3 kernels convolved with the original image to produce gradient approximations
      Edge detection
      • Detect top layer by finding maximum intensity in each column, then black out the region
      • Re-detect the bottom layer by finding maximum intensity
      • Remove outliers from the data in the selected region and fit a surface; output of the surface function gives the depth
