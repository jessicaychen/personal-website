    Project

    The purpose behind the project was: Given a user-selected input of (x,y) coordinates from a top-down view of the retina from an scan, determine the approximate z-coordinate of the bottom layer of the retina through OCT volume scan image segmentation. Bigger purpose: edge detection for robotic retinal injection. I developed an algorithm using MATLAB to detect the bottom edge of the "retina" in a pig-eye model in OCT volume scans. The OCT scans are imported as intensity matrices, and I pre-processed the scans using filters to denoise the scans. A difficulty with this project is the fact that the ROI is the bottom layer, not the top layer. The following techniques were used:

      Pre-processing
      • Normalization - normalize intensity values for later convolution
      • Gaussian blur - blurs an image to reduce noise and detail
      • Median filter - non-linear digital filtering technique that preserves edges while removing noise
      • Convolution - use a kernel to output a image better suited for edge detection
      Edge detection
      • Variation of Dijkstra's Algorithm - finding the shortest path through the highest intensity pixels
      • Polynomial fitting - finding the z-value of maximum intensity in each column and then fitting a 2nd degree curve
