    Posterior Capsule Segmentation

    Advanced Robotic Eye Surgery Lab @ UCLA

    June 2020 - Present

    Background

    Cataracts are known to cause vision impairment and require surgery to replace the lens. In cataract surgery, the cloudy natural cataract lens is phacoemulsified or cut up and removed while an intraocular lens is inserted. While the complication rate is relatively low, complications such as posterior capsular tears may occur. A torn posterior capsule (PC) is dangerous, as it allows for cut up pieces of cataract to potentially fall through into the vitreous cavity. With semi-automated OCT guided cataract removal, it is thus necessary to accurately detect the location of the PC in real time using optical coherence tomography (OCT).

    Project

    I am currently developing an algorithm using MATLAB for the edge detection of the PC in OCT scans. OCT scans are imported as intensity matrices, and the idea is to pre-process the scans using filters to denoise the scans, and then segment the PC using either a shortest-path algorithm or polynomial fitting. A difficulty with this project is the image quality, as these particular scans contain a lot of speckle noise. This project is still a work in progress, but I am so far working with the following techniques:

      Pre-processing
      • Normalization - normalize intensity values for later convolution
      • Gaussian blur - blurs an image to reduce noise and detail
      • Median filter - non-linear digital filtering technique that preserves edges while removing noise
      • Convolution - use a kernel to output a image better suited for edge detection
      Edge detection
      • Variation of Dijkstra's Algorithm - finding the shortest path through the highest intensity pixels
      • Polynomial fitting - finding the z-value of maximum intensity in each column and then fitting a 2nd degree curve
